# Métricas e Metas — Pet Aumigo

## KPIs principais

| Métrica | O que mede | Meta atual |
|---|---|---|
| Pets cadastrados/mês | Volume de uso | Crescimento 20% MoM |
| Taxa de reunificação | Efetividade do produto | >15% dos pets perdidos encontrados |
| Taxa de matching | Qualidade do algoritmo | >60% confidence nos matches |
| Tempo de 1ª notificação | Velocidade | <2 min após cadastro |
| Prestadores ativos | B2B | 50 prestadores aprovados |
| Receita plaquinhas | Monetização | R$3.000/mês |

## Funil atual

```
Visita → Cadastro pet → Matching → Reunificação
         (conversão ~40%)          (meta: 15%)
```

## Canais de aquisição

1. **Orgânico/SEO** — rotas /[tipo]-em-[cidade] (ex: /cachorro-perdido-em-santos)
2. **Grupos WhatsApp** — compartilhamento viral de alertas
3. **Instagram** — conteúdo de reunificações (Hall de Reencontros)
4. **Parceiros** — ONGs e clínicas que indicam a plataforma

## Sazonalidade

- Picos de pets perdidos: Carnaval, Réveillon, temporada de praias (dez-fev), fogos de artifício
- Baixa: maio-agosto

## Dados regionais relevantes

- Santos: maior volume de cadastros
- Guarujá: segundo em volume, muitos pets de turistas
- São Vicente: alta densidade populacional, menor renda (relevante para preço)
- Bertioga: turistas de SP que perdem pets na praia
