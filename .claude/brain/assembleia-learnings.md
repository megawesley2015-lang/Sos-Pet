# Assembleia — Learnings Acumulados

Este arquivo é mantido pelo agente `assembleia`.
Cada descoberta relevante — da internet, de vídeos, de interações — é registrada aqui para persistir entre sessões.

---

## Como usar
- O agente lê este arquivo no início de cada sessão
- Novos learnings são adicionados ao final com data
- Insights que se tornam skills são movidos para `.claude/skills/`
